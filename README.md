# 工业相机选型计算器

机器视觉领域的工业相机选型辅助工具。输入检测需求参数，自动计算分辨率、焦距、曝光时间、带宽、帧率等关键指标，并生成综合推荐方案。

## 功能模块

- **分辨率估算** — 视野(FOV)、检测精度、安全系数 K → 所需像素，推荐相机等级
- **焦距计算** — 工作距离(WD)、视野、传感器靶面 → 推荐焦距、放大倍率、等效35mm
- **曝光时间限值** — 运动速度、像素尺寸、放大倍率 → 最大曝光时间及光源建议
- **带宽需求** — 分辨率、帧率、位深 → 带宽需求，检测 USB 2.0/3.0、GigE、5GigE、Camera Link 适用性
- **最小帧率** — 运动速度、视野、物体长度 → 触发帧率及相机类型建议
- **像素当量** — 像素当量计算，与目标检测精度对比，安全系数评估
- **线阵行频** — 线阵相机场合的行频与像素时钟计算
- **景深估算** — 光圈 F 值和放大倍率估算景深
- **综合推荐** — 汇总所有计算结果，自动生成完整选型方案（含输入参数一览）

## 使用

```bash
# 直接打开或用本地服务器
python -m http.server 8000
# → http://localhost:8000
```

1. 左侧导航栏点击或滚动定位到各模块，填写输入参数，结果自动更新
2. 页面底部 **「刷新推荐方案」** 一键跳转至综合推荐
3. **「打印」** 仅输出推荐方案，适合生成 PDF 报告（白底深色文字，企业级打印格式）
4. 右上角 **主题切换** 按钮切换深色/浅色模式（自动保存偏好）

## 技术栈

纯 HTML/CSS/JS，无第三方依赖。
- CSS 自定义属性实现深色/浅色主题切换（~40 个变量，localStorage 持久化）
- IntersectionObserver 实现滚动导航高亮
- 响应式设计，640px 断点适配移动端
- 打印输出仅含推荐方案，白底深色文字

## 计算参考

- 传感器：1/4″ ~ APS-C（9 种规格）
- 接口带宽上限：USB 2.0(35)、USB 3.0(350)、GigE(100)、5GigE(500)、Camera Link(680) MB/s
- 安全系数 K：1.0 ~ 10.0（步进 0.5）
