# 工业相机选型计算器

用于机器视觉领域的工业相机选型辅助工具。输入检测需求参数，自动计算分辨率、焦距、曝光时间、带宽、帧率等关键指标，并生成综合推荐方案。

## 功能模块

| 模块 | 说明 |
|------|------|
| **分辨率估算** | 根据视野(FOV)、检测精度、安全系数 K 计算所需分辨率，自动推荐相机像素等级 |
| **焦距计算** | 根据工作距离(WD)、视野、传感器靶面计算推荐焦距及放大倍率 |
| **曝光时间限值** | 根据运动速度、像素尺寸、放大倍率计算最大允许曝光时间 |
| **带宽需求** | 根据分辨率、帧率、位深计算带宽，检测各接口(USB 2.0/3.0、GigE、5GigE、Camera Link)适用性 |
| **最小帧率** | 根据运动速度、视野、物体长度计算触发帧率 |
| **像素当量** | 计算像素当量并与目标检测精度对比，给出安全系数评估 |
| **线阵行频** | 线阵相机场合计算所需行频 |
| **景深估算** | 根据光圈 F 值和放大倍率估算景深 |
| **综合推荐** | 汇总上述计算结果，自动生成完整的选型推荐方案 |

## 快速开始

### 方式一：直接打开

用浏览器直接打开 `index.html` 即可使用。

### 方式二：本地服务器（推荐）

```bash
# Python 3
python -m http.server 8000
```

然后在浏览器访问 `http://localhost:8000`。

## 使用说明

1. 左侧导航栏点击对应模块，或滚动页面定位
2. 在各模块中填写或调整输入参数，计算结果自动更新
3. 点击底部 **「刷新推荐方案」** 按钮，跳转至综合推荐方案
4. 点击底部 **「打印」** 可打印或导出为 PDF（仅输出推荐方案）

## 技术栈

- 纯 HTML / CSS / JavaScript，无第三方依赖
- CSS 自定义属性实现深色/浅色主题切换
- IntersectionObserver 实现导航高亮
- localStorage 持久化主题偏好
- 响应式设计，适配桌面与移动端

## 计算参考

- 传感器靶面规格：1/4″ ~ APS-C
- 接口带宽上限：USB 2.0(35 MB/s)、USB 3.0(350 MB/s)、GigE(100 MB/s)、5GigE(500 MB/s)、Camera Link(680 MB/s)
- 安全系数 K 范围：1.0 ~ 10.0
